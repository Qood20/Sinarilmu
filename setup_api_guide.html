Panduan Setup API untuk Sinar Ilmu

Penting: API Key Diperlukan

Aplikasi Sinar Ilmu memerlukan API key yang valid untuk mengakses layanan AI. Fitur analisis materi dan pembuatan soal otomatis memerlukan koneksi ke layanan AI pihak ketiga.

Langkah-langkah Setup API Key

  1. Buka file config/config.php
  2. Temukan baris yang berisi OPENROUTER_API_KEY
  3. Ganti nilai API key dengan API key yang valid dari OpenRouter:
    4. define('OPENROUTER_API_KEY', 'sk-or-v1-YOUR_VALID_API_KEY_HERE');
  4. Simpan perubahan dan restart aplikasi

Cara Mendapatkan API Key OpenRouter

  1. Kunjungi situs https://openrouter.ai/keys
  2. Buat akun atau login ke akun Anda
  3. Buat API key baru
  4. Salin API key dan gunakan di file konfigurasi

Fallback Sistem

Jika API tidak tersedia atau API key tidak valid, sistem akan secara otomatis:

  • Membaca konten dari file yang diupload
  • Membuat soal latihan sederhana berdasarkan isi materi
  • Tetap menyediakan fitur analisis dan latihan soal

Catatan: Meskipun sistem memiliki fallback, penggunaan API key yang valid akan memberikan hasil analisis dan soal latihan yang jauh lebih akurat dan komprehensif.
